# MaskCat
 Makes Hashcat masks from stdin. Format is `MASK:LENGTH:COMPLEXITY`.

 ```
$ echo 'ThisISaT3ST123!' | maskcat
?u?l?l?l?u?u?l?u?d?u?u?d?d?d?s:15:4
 ```

 ```
$ head cracked.lst | maskcat

 ?d?l?d?l?u?d?u?u:8:3
 ```
